新建一个项目

全局安装
cnpm install -g hexo-cli
新建项目初始化
hexo init
运行(按下 ctrl+c 即可停止)
hexo s

hexo 可以用来生成静态的页面，每一个页面都是用makedown来写。

切换主题

切换主题徐亚cd/themes 目录下，再在根目录的_config.yml 修改theme的值
切换主题上传时可以执行 hexo c 清理下目录，然后hexo d 上传
